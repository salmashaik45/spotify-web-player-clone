# Spotify Web Player Clone

A responsive Spotify Web Player UI clone built using HTML and CSS.

## Features
- Sidebar Navigation
- Sticky Header
- Responsive Album Cards
- Music Player UI
- Custom Styled Sliders

## Tech Stack
- HTML5
- CSS3
